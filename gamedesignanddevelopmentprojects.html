SRM Games

SRMGames is a prototype of Game that I have created with the idea of compiling all types of simple games as a subgame to a single game.

Here, players can find different genres of game at the same destination and play whichever game they want to play. I have also made sure to make the game simpler so that a wide range of audience can enjoy playing it and enjoy playing video games. For now, three genres are implemented which are shooting game, riddle game and destructor game.

Castler Game

Castler is a castle demolition game based on the demolition prototype referenced from the book. In this game the main objective is to demolish the castles and cross the levels to get to a new level. The levels here are not connected to each other so a player can play the levels in any order the player wants to play. Also, the shots taken to pass each level are noted on the screen and each level has two sections to pass the level.

The game starts with a start screen then moves forward to a levels pane from which a level can be selected and after the level is passed the level completed screen is shown. To demolish the castles a slingshot mechanism is given which is made up of a catapult and a sphere. Here, the sphere is shot from the catapult and the sphere is hit on the castle to a point marked as goal and if the goal is hit then the castle breaks and gets demolished.

Cherry Scooper Game

CherryScooper is an ApplePicker prototype game. The game's main objective is to collect all the cherries falling from the given tree in the main scene in a basket and the cherry is destroyed or in a sense collected in the basket and this scores points. Through these score points we can also make sure to understand whether the game is able to make new scores and store it or not.

There are a lot of components involved here like a cherry tree, cherries, baskets, highscore counter, a score counter and all the physics and mechanics that help in making the tree move and cherries drop and get collected. The tree is moving in the game as well as the baskets which is making the game have a bit of higher difficulty than the tree just being static there. The mechanics and real game dynamics are explained in further sections.

Whizzle Maze Runner

Maze games or we can say find the path games are games in which a player requires a lot of focus to successfully find the path and to reach a destination with given constraints.

The game has a maze creted whose difficulty increases with level increase and there is a ball with gravity enabled to solve the mazerunner by finding the correct path. The correct path finding uses the path finding algorithm with backtracking to solve the maze crossing puzzle.
